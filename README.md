**PyVessel** PyVessel is a Tauri 2 desktop app with a Vue 3 + TypeScript frontend and a Rust backend. It provides a portable Python runtime manager: users link local Python projects, install managed Python versions (3.10, 3.12, 3.13), and run them in isolated environments with live log streaming.

PyVessel is a desktop app that takes the pain out of Python project management. Link local projects, install managed Python versions via uv, install packages, run scripts with an interactive terminal, and launch AI agents — all in one place. 

Features
- Install managed Python version in a isolated environment
- UV package manager and one-click update
- Copy a Python project to a team-member with modern P2P
- Fire multiple coding agents
- Bultin terminal

Warning: PyVessel is currently in a early beta-release. Expect bugs. Not battle testet accross platforms.
